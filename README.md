

## Math Genius
An application for Grades 1-4 students to learn and practice Math.

###
When cloning, make sure to install all npm packages via npm install.


Application uses mongoDB and user will have to create own db, either locally or on cloud.

.env, ClientId, and MongoDB mappingwill be specific to your use case. Pathways to .env can be manipulated within code based on your specific directory set up.

Runs via node server.js command once DB is set up.
